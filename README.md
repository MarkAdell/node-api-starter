# Node-REST-API-Starter

## Requirements
[Node.js 8](https://nodejs.org/en/)

## Getting Started
Clone the repository, install the dependencies.
```bash
$ git clone git@github.com:MarkAdell/node-api-starter.git

$ cd node-api-starter

$ npm install
```

To run the server
```bash
$ npm run start
```
In watch mode
```bash
$ npm run start:watch
```

Run jest tests
```bash
$ npm run test
```
